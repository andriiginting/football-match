# Football Match
Football match is an application that show data from https://www.thesportsdb.com/api.php using MVP Architecture

# Tech Stack
  - Kotlin
  - Kotlin Coroutines
  - RxJava
  - RxAndroid
  - Anko
  - Glide
  - Sqlite Database
  - Material Design
  - Spek and Mockito (unit testing)
